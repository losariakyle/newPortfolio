Project BaseC – Personal Homelab

Overview

Designed and built a personal homelab environment to experiment with virtualization, networking, and self-hosted services for learning and testing IT infrastructure concepts.

Objectives

Purpose and goals
  • Gain hands-on experience with server management and network configuration.
  • Create a secure, scalable environment for testing automation and deployment workflows.

Technical Setup

Hardware, software, and services used

Hardware

Name Price Purpose

Software & Services (To be developed)

  • Virtualization: Proxmox VE, Docker
  • Services: Pi-hole, Nextcloud, Portainer, Nginx Reverse Proxy
  • Networking: VLAN setup, static IP management, remote access via WireGuard
  • Monitoring: Grafana + Prometheus dashboards

Skills & Outcomes to be developed

Key skills and project results

Key Skills Developed

  • Linux server administration and shell scripting
  • Network configuration and troubleshooting (DHCP, DNS, VLANs)
  • Containerization and virtualization management
  • System monitoring and automation
  • Backup and data redundancy planning

Outcomes

  • Successfully hosting multiple self-managed services accessible over VPN.
  • Improved understanding of DevOps workflows and network security fundamentals.
  • Building a local GitLab instance for version control and CI/CD testing.
