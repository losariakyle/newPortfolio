About DHCP

With dhcp comes a lease time which is the time frame that the device can use the ip address.

After half of the lease time is used the device will ping the server for a new ip address.

If there is no response from the server, the device will continue to use that ip address until the lease time expires.

If devices are frequently connecting and disconnecting from the network then it may be better to have a short lease time

Longer lease times are better for homelab environments as it reduces conflict within the network.

            max-lease-time 7200;
